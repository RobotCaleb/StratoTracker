Live update tracking page for my balloon launch[es]

A micro photo blog of sorts. I will be using it first for live updates of my HAB activities. If the box below is empty either nothing has been posted or the server hasn't told you about any posts yet. In either case, if you leave this page open it will auto-populate when new data comes in.
